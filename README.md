# Controle de Gastos CLI

## Problema
Muitas pessoas têm dificuldade em controlar seus gastos diários.

## Solução
Aplicação em linha de comando para registrar, listar e calcular gastos.

## Funcionalidades
- Adicionar gastos
- Listar gastos
- Ver total

## Tecnologias
- Python

## Instalação
pip install -r requirements.txt

## Execução
python src/app.py

## Testes
pytest

## Lint
flake8 .
